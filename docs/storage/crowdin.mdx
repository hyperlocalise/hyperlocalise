title: "Crowdin adapter"
description: "Configure Crowdin for remote translation synchronization."
---

## Required config

```jsonc
{
  "storage": {
    "adapter": "crowdin",
    "config": {
      "projectID": "123456",
      "apiTokenEnv": "CROWDIN_API_TOKEN"
    }
  }
}
```

## Crowdin Enterprise

Set `apiBaseURL` when your workspace uses Crowdin Enterprise instead of the default Crowdin Cloud API.

```jsonc
{
  "storage": {
    "adapter": "crowdin",
    "config": {
      "projectID": "123456",
      "apiTokenEnv": "CROWDIN_API_TOKEN",
      "apiBaseURL": "https://example.api.crowdin.com/api"
    }
  }
}
```

## Optional config fields

- `apiBaseURL`: Override the Crowdin API base URL. Use this for Crowdin Enterprise.
- `sourceLanguage`
- `targetLanguages`
- `timeoutSeconds`

## Common issues

- token not found: export `CROWDIN_API_TOKEN`
- Enterprise API requests fail: verify `storage.config.apiBaseURL` points to your Enterprise API base URL and uses `https`
- project lookup failure: check project ID and account permissions
- partial updates: review conflict report with `--output json`
