## Repository structure

- `assets/`: static project assets.
- `cmd/`: CLI command definitions, flags, and subcommands.
- `internal/`: internal packages including i18n, sync service, and adapters.
- `scripts/`: helper scripts for project tasks.
- `go.mod`: module dependencies and tool directives.

## Related build files

- `Makefile`: local build, lint, test, and install targets.
- `.goreleaser.yml`: release artifact matrix.
- `.github/workflows/release.yml`: release automation.

## Make targets

```bash
make bootstrap
make check-build
make clean
make cover
make fmt
make lint
make precommit
make run
make staticcheck
make test
```
