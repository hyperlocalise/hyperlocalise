---
title: "Status"
description: "Report translation status by locale, group, and bucket."
---

## Usage

```bash
hyperlocalise status [--config <path>] [flags]
```

Output categories:

- `translated`: non-empty translation value.
- `needs_review`: LLM-generated translation not curated yet.
- `untranslated`: empty translation value.

Flags:

- `--config`: config path, default `i18n.jsonc`.
- `--locale`: locale filter (repeatable).
- `--output`: output format (`csv`).
- `--group`: filter by group name.
- `--bucket`: filter by bucket name.
