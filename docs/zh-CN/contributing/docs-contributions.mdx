## 在GitHub上编辑

1. 打开页面，
2. 点击编辑
3. 提交一个拉取请求。

## 本地文档工作流程

1. 克隆仓库，
2. install Mintlify CLI (`npm i -g mint`),
3. run `mint dev` in `docs/`,
4. validate links with `mint broken-links`,
5. 打开拉取请求。

## 写作标准

- 使用主动语态和第二人称。
- 请保持示例代码可执行。
- 在每页开头注明目标和先决条件。
- 确保命令标志与当前 CLI 帮助输出保持一致。
