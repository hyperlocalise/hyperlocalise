## 在 GitHub 上编辑

1. 打开页面，
2. 单击编辑，
3. 提交一个拉取请求。

## 本地文档工作流

1. 克隆存储库，
2. 安装 Mintlify CLI (`npm i -g mint`),
3. 运行 `mint dev` in `docs/`,
4. 使用以下内容验证链接 `mint broken-links`,
5. 打开拉取请求。

## 写作标准

- 使用主动语态和第二人称，
- 保持示例可执行，
- 在每页开头写明目标和先决条件，
- 使命令标志与当前 CLI 帮助输出保持一致。
