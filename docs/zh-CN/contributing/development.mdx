## 先决条件

- Go 1.26+
- `git`
- `make` available in your shell

## 克隆仓库

```bash
git clone https://github.com/quiet-circles/hyperlocalise.git
cd hyperlocalise
```

## Bootstrap 工具和模块

```bash
make bootstrap
```

这会下载 Go 模块并安装指定的 lint 工具链。

## 每日开发循环

```bash
make fmt
make lint
make test
make check-build
```

## 在本地运行 CLI

```bash
make run
# or
go run -ldflags "-X main.version=dev" main.go
```

## 安装本地二进制文件

```bash
make install
hyperlocalise version
```

## 关键代码位置

- `cmd/`: Cobra commands, flags, and CLI output behavior
- `internal/config/`：`i18n.jsonc` 的解析和验证
- `internal/i18n/runsvc/`：`run` 规划和执行流程
- `internal/i18n/syncsvc/`: sync orchestration and conflict logic
- `internal/i18n/storage/`: 适配器合同和提供商集成

## 在提交 PR 之前

```bash
make precommit
```

这在与贡献者相同的流程中执行格式化、代码检查、测试和构建检查。
