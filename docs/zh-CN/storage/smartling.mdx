---
title: "Smartling 适配器"
description: "配置 Smartling 以进行远程翻译拉取/推送同步。"
---

## 必需配置

```jsonc
{
  "storage": {
    "adapter": "smartling",
    "config": {
      "projectID": "your-project-id",
      "userIdentifier": "your-smartling-user-identifier",
      "userSecretEnv": "SMARTLING_USER_SECRET"
    }
  }
}
```

## 必需的密钥

```bash
export SMARTLING_USER_SECRET="your-smartling-user-secret"
```

`userSecret` 不支持在配置中内联。请仅使用环境变量。

## 可选配置字段

- `targetLanguages`
- `timeoutSeconds` (默认为 `30`)

## 常见问题

- 缺少密钥：导出 `SMARTLING_USER_SECRET` 在同一个 Shell 会话中
- 缺少用户标识符：已设置 `storage.config.userIdentifier`
- 区域设置不匹配：对齐 `targetLanguages` 使用你的项目区域设置
