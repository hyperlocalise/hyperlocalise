title: "LILT AI 适配器"
description: "配置 LILT AI 以进行远程翻译的拉/推同步。"
---

## 所需配置

```jsonc
{
  "storage": {
    "adapter": "lilt",
    "config": {
      "projectID": "123",
      "apiTokenEnv": "LILT_API_TOKEN"
    }
  }
}
```

## 令牌解析

1. 环境变量名为`storage.config.apiTokenEnv`
2. fallback: `LILT_API_TOKEN`

`apiToken` inline in config is not supported. Use environment variables only.

## 可选配置字段

- `targetLanguages`
- `timeoutSeconds` (defaults to `30`)
- `pollIntervalMs` (defaults to `1000`)
- `maxPolls` (defaults to `60`)

## 拉取行为

- 为每个语言环境启动一个导出任务
- 调查结果导出状态，直到任务完成
- 下载 JSON 格式的工件 (单个文件或 ZIP 压缩包)
- 规范条目，并跳过空键/语言/值行

## 常见问题

- 缺少 token: export `LILT_API_TOKEN` 在同一终端会话中
- 缺少项目ID：设置`storage.config.projectID`
- timeout or slow export: increase `timeoutSeconds`, `pollIntervalMs`, or `maxPolls`
