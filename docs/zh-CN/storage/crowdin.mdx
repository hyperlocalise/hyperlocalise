---
title: "Crowdin 适配器"
description: "配置 Crowdin 以进行远程翻译同步。"
---

## 所需配置

```jsonc
{
  "storage": {
    "adapter": "crowdin",
    "config": {
      "projectID": "123456",
      "apiTokenEnv": "CROWDIN_API_TOKEN"
    }
  }
}
```

## 可选配置字段

- `sourceLanguage`
- `targetLanguages`
- `timeoutSeconds`

## 常见问题

- token not found: export `CROWDIN_API_TOKEN`
- 项目查找失败：请检查项目 ID 和账户权限
- partial updates: review conflict report with `--output json`
