---
title: "POEditor 适配器"
description: "配置 POEditor 以进行同步拉取和同步推送。"
---

## 必需配置

```jsonc
{
  "storage": {
    "adapter": "poeditor",
    "config": {
      "projectID": "123456",
      "apiTokenEnv": "POEDITOR_API_TOKEN"
    }
  }
}
```

## 令牌解析

1. 由以下内容命名的环境变量 `storage.config.apiTokenEnv`
2. 回退： `POEDITOR_API_TOKEN`

## 可选配置字段

- `sourceLanguage`
- `targetLanguages`
- `timeoutSeconds`

## 常见问题

- 无效令牌：请确认环境变量已在当前 Shell 中导出
- 项目 ID 错误：请确认来自 POEditor 的数字项目 ID
- 缺少的区域设置：请确保 `targetLanguages` 与配置目标一致
