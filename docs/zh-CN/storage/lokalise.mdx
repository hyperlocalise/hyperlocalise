---
title: "本地化适配器"
description: "配置 Lokalise API v2 以进行拉/推同步。"
---

## 必需的配置

```jsonc
{
  "storage": {
    "adapter": "lokalise",
    "config": {
      "projectID": "your-project-id",
      "apiTokenEnv": "LOKALISE_API_TOKEN"
    }
  }
}
```

## 可选配置字段

- `targetLanguages`
- `timeoutSeconds`

## Token 解析

- 默认环境变量：`LOKALISE_API_TOKEN`
- override with `apiTokenEnv`

## 常见问题

- 未授权错误：缺少或权限不足的令牌
- 本地化不匹配：将 Lokalise 中的语言代码与目标本地化相对应
- timeout errors: increase `timeoutSeconds` for large projects
