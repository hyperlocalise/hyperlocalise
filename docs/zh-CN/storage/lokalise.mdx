---
title: "Lokalise 适配器"
description: "配置 Lokalise API v2 以进行拉取/推送同步。"
---

## 必需配置

```jsonc
{
  "storage": {
    "adapter": "lokalise",
    "config": {
      "projectID": "your-project-id",
      "apiTokenEnv": "LOKALISE_API_TOKEN"
    }
  }
}
```

## 可选配置字段

- `targetLanguages`
- `timeoutSeconds`

## 令牌解析

- 默认环境变量：`LOKALISE_API_TOKEN`
- 用`apiTokenEnv`覆盖

## 常见问题

- 未授权错误：令牌缺失或权限范围不足
- 语言区域不匹配：将 Lokalise 语言代码与目标区域设置对齐
- 超时错误：对于大型项目，请增加`timeoutSeconds`
