## 目标

Create an `i18n.jsonc`, run translation planning safely, generate outputs, and review status.

## 1. 初始化配置

```bash
hyperlocalise init
```

This writes `i18n.jsonc` in your current directory.

## 2. 配置提供商凭据

## 3. 先进行测试

```bash
hyperlocalise run --dry-run
```

检查 `planned_total`、`executable_total`以及列出的 `tasks`。

## 4. 执行生成

```bash
hyperlocalise run
```

This writes translations and updates `.hyperlocalise.lock.json`. For details, see [Lockfile contract](/reference/lockfile-contract).

## 5. 检查状态

```bash
hyperlocalise status --output csv
```

Use `needs_review` rows to identify entries that still need human curation.

## 6. 根据需要评估翻译质量

```bash
hyperlocalise eval run --eval-set ./eval-set.json --output ./eval-report.json
```

使用此功能，在比较提示/个人资料/提供商选项或在推广大型翻译更新之前进行操作。
