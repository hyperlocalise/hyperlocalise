## 先决条件

- macOS 或 Linux
- one of: installer script, prebuilt binary, `.deb`/`.rpm` package, or Go toolchain

## 使用引导脚本进行安装

使用稳定的 bootstrap URL

```bash
curl -fsSL https://raw.githubusercontent.com/hyperlocalise/hyperlocalise/main/install.sh | bash
```

固定特定版本：

```bash
curl -fsSL https://raw.githubusercontent.com/hyperlocalise/hyperlocalise/main/install.sh | VERSION=v1.2.3 bash
```

安装脚本也已附加到每个 GitHub 发布的资产中，这样您就可以直接从发布资产查看或下载标记版本。

## 从源代码安装 （Go）

```bash
go install github.com/hyperlocalise/hyperlocalise@latest
```

## 从仓库构建

```bash
make bootstrap
make install
```

## 验证安装

```bash
hyperlocalise version
hyperlocalise --help
```

预期的输出包括以下命令：`init`、`run`、`status`、`sync`、`version`以及`update`。

## 安装 Hyperlocalise 技能 (skills.sh)

安装：

```bash
npx skills add https://github.com/hyperlocalise/hyperlocalise --skill hyperlocalise
```

这使用 [skills.sh](https://skills.sh)安装脚本，通过`npx` 运行。

## 升级

使用内置的自动更新命令：

```bash
hyperlocalise update
```

指定版本：

```bash
hyperlocalise update v1.2.3
```

您仍然可以选择重新运行您原始的安装方法。
