## 规范文件名称

锁文件名称已固定为 `.hyperlocalise.lock.json`。

`hyperlocalise` does not currently read `i18n.lock` or other alias names.

Keep the file in your project root so `run` and sync commands share the same checkpoint state.

## 锁文件的作用是什么

将锁文件视为一个**建议性的检查点**，而不是您的最终来源。

- 翻译内容来源的真相是您的源文件。
- 生成输出的最终来源是您的目标文件。
- 用于同步远程状态的唯一来源是您的 TMS 加上本地文件。

锁文件仅记录 CLI 已经尝试或完成的内容，以便避免重复工作。

## 模式

当前形状：

```json
{
  "adapter": "crowdin",
  "project_id": "12345",
  "last_pull_at": "2026-01-15T12:34:56Z",
  "locale_states": {
    "es-ES": {
      "revision": "rev_abc123",
      "updated_at": "2026-01-15T12:00:00Z"
    }
  },
  "run_completed": {
    "apps/web/lang/es-ES.json::checkout.submit": {
      "source_hash": "7f4f7d..."
    }
  }
}
```

字段含义：

- `adapter`: active TMS adapter name.
- `project_id`: adapter-specific project identifier.
- `last_pull_at`: timestamp of the latest successful sync pull.
- `locale_states`: 针对本地化的检查点地图。
  - `revision`: adapter revision or cursor for incremental pull logic.
  - `updated_at`: last time that locale checkpoint was refreshed.
- `run_completed`：用于`run`的按任务完成情况图，用于跳过决策。
  - `source_hash`: 在完成时捕获的源指纹。

## 通过命令控制生命周期

### `hyperlocalise run`

- 读取`.hyperlocalise.lock.json`以进行任务规划。
- Skips work already marked complete in `run_completed`.
- 将成功的任务回写到`run_completed`。

### `hyperlocalise sync pull`

- 读取现有适配器和区域设置检查点元数据。
- Updates `adapter`, `project_id`, `last_pull_at`, and `locale_states` after successful pull.

### `hyperlocalise sync push`

- 可能从锁文件中读取适配器/项目的上下文。
- 不取代源/目标内容的权威性。

## 安全重置指南

当检查点状态陈旧或不再可信时，重置锁文件。

常见重置场景：

- 你对原始文本的结构进行了大幅修改
- 您修改了存储桶映射或文件路径模板
- 您更改了适配器或项目
- 您怀疑存在损坏或手动编辑过的检查点条目

安全重置步骤：

1. 请确保您的源文件和生成的文件的状态已提交或备份。
2. 删除 `.hyperlocalise.lock.json`
3. run `hyperlocalise run --dry-run` to inspect the larger plan
4. 运行`hyperlocalise run` 和/或 `hyperlocalise sync pull`以重建检查点状态

重置后预期的副作用：

- `run` may re-execute tasks that were previously skipped
- 下一次同步拉取可能会执行更广泛的本地化刷新
- CLI 运行时可能会暂时增加，直到检查点被重建

## 团队会议

- 除非在调试本地问题，否则不要手动编辑锁文件字段。
- 在 CI 脚本中明确处理锁文件。
- 在合并请求中检查 `lockfile` 更改，以便在/同步行为出现意外时进行审查。
