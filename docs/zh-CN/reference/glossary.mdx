---
title: "术语表"
description: "跨区域配置和工作流程中使用的通用术语。"
---

- `bucket`: reusable file mapping set that defines `from`/`to` templates.
- `group`: 定义执行范围，将目标区域与名称绑定。
- `profile`: LLM 提供商/模型/提示定义。
- `rule`：基于优先级的从组到用户档案的映射。
- `draft`: translation generated locally by LLM workflow.
- `curated`: translation reviewed by a human in remote TMS.
- `provenance`: metadata describing origin/state and update history.
- `sidecar metadata`: separate metadata file that stores operational state.
- `conflict`: value mismatch requiring explicit resolution policy.
