---
title: "状态"
description: "以 CSV 格式按 key/namespace/locale 报告翻译状态。"
---

## 用法

```bash
hyperlocalise status [--config <path>] [flags]
```

## 标志

- `--config`：配置文件路径
- `--locale`: 目标区域设置筛选器 (可重复的)
- `--output`：输出格式 (`csv`)
- `--group`：按组名筛选
- `--bucket`：按存储桶名称筛选

## 状态值

- `translated`：值为 non-为空且未标记为草稿
- `needs_review`：provenance 为 `origin=llm` 和 `state=draft`
- `untranslated`：值为空

## 示例

```bash
hyperlocalise status --config i18n.jsonc --locale es-ES --output csv
```
