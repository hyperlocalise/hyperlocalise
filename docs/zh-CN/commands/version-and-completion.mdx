---
title: "版本, 更新, 和完成"
description: "检查已安装的版本，自动更新 CLI，并生成 shell 补全脚本。"
---

## 版本

```bash
hyperlocalise version
```

预期的输出格式：

```text
hyperlocalise: <version>
```

## 更新

更新到最新发布的版本：

```bash
hyperlocalise update
```

固定特定版本标签：

```bash
hyperlocalise update v1.2.3
```

## 完成

使用内置的 Cobra 补全命令：

```bash
hyperlocalise completion --help
```

然后生成并安装用于您的 shell 的补全脚本。
