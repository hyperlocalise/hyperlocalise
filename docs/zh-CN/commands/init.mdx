---
title: "初始化"
description: "在您的项目中创建一个或覆盖 i18n.jsonc 模板。"
---

## 用法

```bash
hyperlocalise init [--force]
```

## 它做什么

- writes `i18n.jsonc` from the built-in template
- fails if file already exists (unless `--force`)

## 标志

- `--force`: overwrite an existing `i18n.jsonc`

## 典型流程

```bash
hyperlocalise init
hyperlocalise run --dry-run
```
