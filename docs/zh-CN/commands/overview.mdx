## CLI 形状

```bash
hyperlocalise [command]
```

主要命令：

- `init`
- `run`
- `eval`
- `status`
- `sync pull`
- `sync push`
- `version`
- `completion`

- 拉取精选翻译： [同步拉取](/commands/sync-pull)
- 推送本地更改： [同步推送](/commands/sync-push)
