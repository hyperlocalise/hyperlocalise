## 解决冲突

典型情况：本地精选值与远程值不一致。

安全分辨率：

1. 运行`sync pull` 使用`--output json`
2. 检查冲突条目
3. 确定唯一来源
4. 重新运行，使用更新后的本地/远程状态

`--apply-curated-over-draft` only applies when local entry is draft.

## 冲突

典型情况：本地草稿与远程精选值不一致。

安全分辨率：

1. 检查冲突报告，格式为 JSON
2. 保持默认冲突失败行为
3. use `--force-conflicts` only for reviewed exceptions

## 推荐默认值

- 保留`--fail-on-conflict=true`
- 保持 `--dry-run=true`，直到报告被审核
- 在 CI 中存档 JSON 报告以创建审计日志
