---
title: "Ollama"
description: "将 Ollama 配置为用于生成翻译的 LLM 提供程序。"
---

## 提供者值

在 `llm.profiles.<name>.provider` 中使用 `ollama`。

## 示例资料片段

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "ollama",
        "model": "qwen2.5:7b",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

## 可选环境变量

`OLLAMA_BASE_URL` 默认为 `http://127.0.0.1:11434/v1`。

`OLLAMA_API_KEY` 默认为 `ollama`。
