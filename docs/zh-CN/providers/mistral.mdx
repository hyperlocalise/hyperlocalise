---
title: "Mistral"
description: "配置 hyperlocalise 以使用 Mistral 的与 OpenAI 兼容的 API。"
---

使用`mistral` 在 `llm.profiles.<name>.provider` 中。

## 配置示例

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "mistral",
        "model": "mistral-large-latest",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

## 环境变量

```bash
# Optional, defaults to https://api.mistral.ai/v1
export MISTRAL_BASE_URL="https://api.mistral.ai/v1"

export MISTRAL_API_KEY="your-mistral-api-key"
```
