---
title: "Mistral"
description: "配置 hyperlocalise 以使用 Mistral 的 OpenAI 兼容 API。"
---

在 `llm.profiles.<name>.provider` 中使用 `mistral`。

## 配置示例

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "mistral",
        "model": "mistral-large-latest",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

## 必需的环境变量

<CodeGroup>
```bash Export
export MISTRAL_API_KEY="your-mistral-api-key"
```

```bash .env.local
MISTRAL_API_KEY="your-mistral-api-key"
```
</CodeGroup>

## 可选环境变量

`MISTRAL_BASE_URL` 默认为 `https://api.mistral.ai/v1`。
