---
title: "OpenAI"
description: "将 OpenAI 配置为用于生成翻译的 LLM 提供方。"
---

## 提供商值

使用 `openai` in `llm.profiles.<name>.provider`.

## 示例配置文件

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "openai",
        "model": "gpt-5.2",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

## 必需的环境变量

<CodeGroup>
```bash Export
export OPENAI_API_KEY="your-openai-api-key"
```

```bash .env.local
OPENAI_API_KEY="your-openai-api-key"
```
</CodeGroup>
