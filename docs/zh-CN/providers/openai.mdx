---
title: "OpenAI"
description: "将 OpenAI 配置为用于翻译生成的 LLM 提供商。"
---

## 提供商价值

使用 `openai` 在 `llm.profiles.<name>.provider` 中。

## 例子资料

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "openai",
        "model": "gpt-5.2",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

## 所需环境变量

<CodeGroup>
```bash Export
export OPENAI_API_KEY="your-openai-api-key"
```

```bash .env.local
OPENAI_API_KEY="your-openai-api-key"
```
</CodeGroup>
