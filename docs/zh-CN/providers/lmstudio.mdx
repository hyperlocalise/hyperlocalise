---
title: "LM Studio"
description: "将 LM Studio 配置为用于生成翻译的 LLM 提供程序。"
---

## 提供者值

在 `llm.profiles.<name>.provider` 中使用 `lmstudio`。

## 示例简介

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "lmstudio",
        "model": "qwen2.5-7b-instruct",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

## 可选环境变量

`LM_STUDIO_BASE_URL` 默认为 `http://127.0.0.1:1234/v1`。

`LM_STUDIO_API_KEY` 默认为 `lm-studio`。
