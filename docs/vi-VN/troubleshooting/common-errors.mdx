## `i18n.jsonc already exists; use --force to overwrite`

Cause: running `init` where config already exists.

Sửa

```bash
hyperlocalise init --force
```

## `storage config is required`

Cause: `sync` command used without top-level `storage` config.

Sửa: thêm `storage.adapter` và `storage.config` vào `i18n.jsonc`.

## `unsupported output format`

Cause: wrong `--output` value.

Sửa lỗi:

- `status`: only `csv`
- `sync`: `text` hoặc `json`

## `unsupported file extension` during `run`

Nguyên nhân: một tệp nguồn hoặc đích được chỉ định sử dụng phần mở rộng không được trình xử lý cục bộ hỗ trợ.

Sửa lỗi:

- use one of: `.json`, `.arb`, `.xlf`, `.xliff`, `.po`, `.html`, `.md`, `.mdx`, `.strings`, `.stringsdict`, `.csv`
- verify bucket `from` and `to` paths resolve to the expected file names

## `unsupported file extension` trong quá trình `run`

Nguyên nhân: [1mToken[0m môi trường bị thiếu/không hợp lệ.

Khắc phục: xuất lại mã thông báo trong cùng một phiên và chạy lại.
