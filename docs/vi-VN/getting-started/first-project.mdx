## Mẫu cấu hình tối thiểu

Sử dụng một nhóm và một bộ để bắt đầu:

- `locales.source`: your source locale (for example `en-US`)
- `locales.targets`: one target locale (for example `es-ES`)
- `buckets.ui.files`: map source path template to target path template
- `groups.default`: attach `targets` and `buckets`

For full fields, see [i18n config](/configuration/i18n-config).

In this setup, translations are sourced from `packages/ui/lang` and generated into `apps/web/lang`.

## Các đối tượng được tạo

Sau `hyperlocalise run`:

- translated target files under your configured `to` paths
- `.hyperlocalise.lock.json` in project root
- tệp dữ liệu phụ trợ cho việc theo dõi nguồn gốc, tùy thuộc vào cấu trúc lưu trữ của bạn
