## Yêu cầu

- macOS hoặc Linux
- một trong: Installer script, pre-built binary, gói `.deb`/`.rpm`, hoặc Go toolchain

## Cài đặt bằng script khởi tạo

Sử dụng URL bootstrap ổn định:

```bash
curl -fsSL https://hyperlocalise.com/install | bash
```

Gắn một phiên bản cụ thể:

```bash
curl -fsSL https://hyperlocalise.com/install | VERSION=v1.2.3 bash
```

"Script cài đặt cũng được đính kèm với mỗi bản phát hành trên GitHub, vì vậy bạn có thể xem hoặc tải xuống phiên bản được đánh dấu trực tiếp từ tài sản của bản phát hành.

## Cài đặt từ nguồn (Go)

```bash
go install github.com/hyperlocalise/hyperlocalise@latest
```

## Xây dựng từ kho lưu trữ

```bash
make bootstrap
make install
```

## Kiểm tra cài đặt

```bash
hyperlocalise version
hyperlocalise --help
```

Kết quả dự kiến bao gồm các lệnh sau: `init`, `run`, `status`, `sync`, `version` và `update`.

## Cài đặt kỹ năng 'Hyperlocalise' (skills.sh)

Cài đặt:

```bash
npx skills add https://github.com/hyperlocalise/hyperlocalise --skill hyperlocalise
```

This uses the [skills.sh](https://skills.sh) installer via `npx`.

## Cập nhật

Sử dụng lệnh tự cập nhật tích hợp:

```bash
hyperlocalise update
```

Gắn một phiên bản cụ thể:

```bash
hyperlocalise update v1.2.3
```

Bạn vẫn có thể thử lại phương pháp cài đặt ban đầu nếu muốn.
