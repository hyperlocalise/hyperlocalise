## Điều kiện tiên quyết

- macOS hoặc Linux
- một trong các loại sau: script cài đặt, tệp nhị phân dựng sẵn, gói `.deb`/`.rpm`, hoặc bộ công cụ Go

## Cài đặt bằng script bootstrap

Sử dụng URL bootstrap ổn định:

```bash
curl -fsSL https://hyperlocalise.com/install | bash
```

Ghim một bản phát hành cụ thể:

```bash
curl -fsSL https://hyperlocalise.com/install | VERSION=v1.2.3 bash
```

Khi bạn mở `https://hyperlocalise.com/install` trong trình duyệt, nó sẽ chuyển hướng đến trang này. Các ứng dụng dòng lệnh như `curl` sẽ nhận luồng kịch bản cài đặt thay vào đó.

Script cài đặt cũng được đính kèm vào từng bản phát hành GitHub, vì vậy bạn có thể xem lại hoặc tải trực tiếp phiên bản được gắn thẻ từ các tài sản phát hành.

## Cài đặt từ mã nguồn (Go)

```bash
go install github.com/hyperlocalise/hyperlocalise@latest
```

## Xây dựng từ kho lưu trữ

```bash
make bootstrap
make install
```

## Xác minh cài đặt

```bash
hyperlocalise version
hyperlocalise --help
```

Đầu ra dự kiến bao gồm các lệnh khả dụng: `init`, `run`, `status`, `sync`, `version`, và `update`.

## Cài đặt kỹ năng Hyperlocalise (skills.sh)

Cài đặt:

```bash
npx skills add https://github.com/hyperlocalise/hyperlocalise --skill hyperlocalise
```

Bản này sử dụng trình cài đặt [skills.sh](https://skills.sh) thông qua `npx`.

## Nâng cấp

Sử dụng lệnh tự cập nhật tích hợp sẵn:

```bash
hyperlocalise update
```

Ghim một bản phát hành cụ thể:

```bash
hyperlocalise update v1.2.3
```

Bạn vẫn có thể chạy lại phương pháp cài đặt ban đầu nếu muốn.
