## Điều kiện tiên quyết

- macOS hoặc Linux
- một trong các: tập lệnh trình cài đặt, tệp nhị phân dựng sẵn, `.deb`/`.rpm` gói hoặc bộ công cụ Go

## Cài đặt bằng script bootstrap

Sử dụng URL bootstrap ổn định:

```bash
curl -fsSL https://raw.githubusercontent.com/quiet-circles/hyperlocalise/main/install.sh | bash
```

Ghim một bản phát hành cụ thể:

```bash
curl -fsSL https://raw.githubusercontent.com/quiet-circles/hyperlocalise/main/install.sh | VERSION=v1.2.3 bash
```

Tập lệnh cài đặt cũng được đính kèm trong mỗi bản phát hành trên GitHub để bạn có thể xem lại hoặc tải xuống trực tiếp phiên bản theo thẻ từ tài nguyên của bản phát hành.

## Cài đặt từ mã nguồn (Đi)

```bash
go install github.com/quiet-circles/hyperlocalise@latest
```

## Xây dựng từ kho lưu trữ

```bash
make bootstrap
make install
```

## Xác minh cài đặt

```bash
hyperlocalise version
hyperlocalise --help
```

Kết quả mong đợi bao gồm các lệnh khả dụng: `init`, `run`, `status`, `sync`, và `version`.

## Cài đặt kỹ năng Hyperlocalise (skills.sh)

Cài đặt:

```bash
npx skills add https://github.com/quiet-circles/hyperlocalise --skill hyperlocalise
```

Phần này sử dụng [skills.sh](https://skills.sh) trình cài đặt qua `npx`.

## Nâng cấp

Về việc-chạy phương thức cài đặt của bạn.

- Người dùng trình cài đặt: chạy lại lệnh bootstrap, tùy chọn với `VERSION=<tag>`.
- Để cài đặt người dùng Go: chạy `go install ...@latest` lần nữa.
- Người dùng kho lưu trữ: kéo các thay đổi mới nhất, sau đó chạy `make install`.
