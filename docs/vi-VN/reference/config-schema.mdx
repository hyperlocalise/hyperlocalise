## Chỉ khóa cấp cao

- `locales`
- `buckets`
- `groups`
- `llm`
- `storage` (required for `sync`)
- `cache` (tùy chọn)

## `locales`

- `source` (chuỗi, bắt buộc)
- `targets` (mảng chuỗi, bắt buộc)
- `fallbacks` (đối tượng, tùy chọn)

## `buckets`

- bản đồ tên bucket -> định nghĩa bucket
- `files` array with `from` and `to` templates

## `groups`

- bản đồ tên nhóm -> định nghĩa nhóm
- `targets` mảng
- `buckets` array

## `llm`

- `profiles` map of profile name -> provider/model/prompt
- tùy chọn `context_memory` đối tượng với `provider` và `model` để thay thế ngữ cảnh
- các nhà cung cấp được hỗ trợ: `openai` | `azure_openai` | `anthropic` | `gemini` | `bedrock` | `lmstudio` | `groq` | `mistral` | `ollama`
- `rules` array with `priority`, `group`, `profile`

## `storage`

- `adapter`: `poeditor` | `lilt` | `lokalise` | `crowdin` | `phrase` | `smartling`
- `config`: trường cụ thể cho thiết bị

## `cache`

- `enabled` boolean
- `db_path` chuỗi
- `sqlite.max_open_conns` integer
- `sqlite.max_idle_conns` số nguyên
- `sqlite.conn_max_lifetime_seconds` integer
- `l1.enabled` boolean
- `l1.max_items` integer
- `l2.enabled` boolean
- `l2.auto_accept_threshold` số trong `[0,1]`
- `rag.enabled` kiểu dữ liệu boolean
- `rag.top_k` integer

Giá trị siêu dữ liệu TM được sử dụng bởi bộ nhớ cache:

- nguồn gốc: `curated` | `draft` | `llm` | `tms` | `unknown`
- source: `run` | `sync_pull` | `sync_push` | `manual` | `import` | `legacy` | `unknown`

Ví dụ, vui lòng xem [cấu hình i18n](/configuration/i18n-config).


## Ghi chú quy trình CSV

Use `buckets.*.files[].from` and `to` to point at `.csv` files when you run CSV-based localization flows.

- Per-locale files typically use `[locale]` in the `to` path.
- Shared CSV files can reuse the same `from` and `to` path.
