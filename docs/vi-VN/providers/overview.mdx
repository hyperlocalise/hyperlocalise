## Các trường bắt buộc

Each `llm.profiles.<name>` entry requires:

- `provider`
- `model`

Trường profile tùy chọn:

- `system_prompt`
- `user_prompt`
- `prompt` (deprecated)

## Biến Prompt

These placeholders are available in `prompt`, `system_prompt`, and `user_prompt`:

- `{{source}}`
- `{{target}}`
- `{{input}}`
