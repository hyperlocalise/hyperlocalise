---
title: "Mistral"
description: "Cấu hình hyperlocalise để sử dụng API tương thích với OpenAI của Mistral."
---

Sử dụng `mistral` trong `llm.profiles.<name>.provider`.

## Ví dụ cấu hình

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "mistral",
        "model": "mistral-large-latest",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

## Biến môi trường bắt buộc

<CodeGroup>
```bash Export
export MISTRAL_API_KEY="your-mistral-api-key"
```

```bash .env.local
MISTRAL_API_KEY="your-mistral-api-key"
```
</CodeGroup>

## Biến môi trường tùy chọn

`MISTRAL_BASE_URL` mặc định là `https://api.mistral.ai/v1`.
