---
title: "Mistral"
description: "Cấu hình hyperlocalise để sử dụng API tương thích với OpenAI của Mistral."
---

Use `mistral` in `llm.profiles.<name>.provider`.

## Ví dụ cấu hình

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "mistral",
        "model": "mistral-large-latest",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

## Biến môi trường

```bash
# Optional, defaults to https://api.mistral.ai/v1
export MISTRAL_BASE_URL="https://api.mistral.ai/v1"

export MISTRAL_API_KEY="your-mistral-api-key"
```
