## Giá trị nhà cung cấp

Sử dụng HLMDPH_10DE9655C125_0 trong HLMDPH_31D40B49F448_1.

## Ví dụ về hồ sơ

```json
{
  "llm": {
    "profiles": {
      "default": {
        "provider": "azure_openai",
        "model": "gpt-4.1-mini",
        "prompt": "Translate from {{source}} to {{target}}:\n\n{{input}}"
      }
    }
  }
}
```

## Các biến môi trường cần thiết
