## Cấu hình bắt buộc

```jsonc
{
  "storage": {
    "adapter": "crowdin",
    "config": {
      "projectID": "123456",
      "apiTokenEnv": "CROWDIN_API_TOKEN"
    }
  }
}
```

## Các trường cấu hình tùy chọn

- `sourceLanguage`
- `targetLanguages`
- `timeoutSeconds`

## Các vấn đề phổ biến

- token not found: export `CROWDIN_API_TOKEN`
- thất bại khi tìm kiếm dự án: kiểm tra ID dự án và quyền của tài khoản
