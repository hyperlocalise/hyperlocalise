title: "Bộ chuyển đổi Crowdin"
description: "Cấu hình Crowdin để đồng bộ hóa bản dịch từ xa."
---

## Cấu hình bắt buộc

```jsonc
{
  "storage": {
    "adapter": "crowdin",
    "config": {
      "projectID": "123456",
      "apiTokenEnv": "CROWDIN_API_TOKEN"
    }
  }
}
```

## Các trường cấu hình tùy chọn

- `sourceLanguage`
- `targetLanguages`
- `timeoutSeconds`

## Các vấn đề thường gặp

- không tìm thấy token: export `CROWDIN_API_TOKEN`
- không thể tra cứu dự án: kiểm tra mã dự án và quyền tài khoản
- cập nhật một phần: xem xét báo cáo xung đột với `--output json`
