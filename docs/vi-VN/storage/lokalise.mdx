## Cấu hình bắt buộc

```jsonc
{
  "storage": {
    "adapter": "lokalise",
    "config": {
      "projectID": "your-project-id",
      "apiTokenEnv": "LOKALISE_API_TOKEN"
    }
  }
}
```

## Các trường cấu hình tùy chọn

- `targetLanguages`
- `timeoutSeconds`

## Giải quyết token

- biến môi trường mặc định: `LOKALISE_API_TOKEN`
- thay thế bằng HLMDPH_92EE6E77CFF7_0

## Các vấn đề thường gặp

- Lỗi không được phép: token bị thiếu hoặc phạm vi không đủ
- Không khớp ngôn ngữ: Đảm bảo mã ngôn ngữ Lokalise khớp với các ngôn ngữ đích
- lỗi hết thời gian: tăng HLMDPH_4C5BB18D61C4_0 cho các dự án lớn
