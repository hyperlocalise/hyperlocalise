## Cấu hình bắt buộc

```jsonc
{
  "storage": {
    "adapter": "phrase",
    "config": {
      "projectID": "project-uuid",
      "apiTokenEnv": "PHRASE_API_TOKEN",
      "mode": "strings"
    }
  }
}
```

## Các trường cấu hình tùy chọn

- `sourceLanguage`
- `targetLanguages`
- `timeoutSeconds`
- `fileFormat` (bắt buộc khi `mode` là `files`)

## Chế độ

- `strings`: kéo và đẩy bằng cách sử dụng Phrase key và các API dịch.
- `files`: kéo và đẩy bằng các tác vụ xuất/nhập Phrase.

## Các vấn đề thường gặp

- không tìm thấy token: export `PHRASE_API_TOKEN`
- lỗi xác thực chế độ: đặt `mode` đến `strings` hoặc `files`
- Lỗi nhập chế độ tệp: xác minh `fileFormat` khớp với cài đặt dự án của bạn
