---
title: "Phrase adapter"
description: "Cấu hình Phrase để đồng bộ hóa bản dịch từ xa."
---

## Cấu hình bắt buộc

```jsonc
{
  "storage": {
    "adapter": "phrase",
    "config": {
      "projectID": "project-uuid",
      "apiTokenEnv": "PHRASE_API_TOKEN",
      "mode": "strings"
    }
  }
}
```

## Các trường cấu hình tùy chọn

- `sourceLanguage`
- `targetLanguages`
- `timeoutSeconds`
- `fileFormat` (required when `mode` is `files`)

## Các chế độ

- `strings`: pull and push using Phrase key and translation APIs.
- `files`: pull and push using Phrase export/import jobs.

## Các vấn đề thường gặp

- token not found: export `PHRASE_API_TOKEN`
- lỗi xác thực chế độ: đặt HLMDPH_58CE2FBCEBC2_0 hoặc HLMDPH_5DD0DAE7EBCC_1 hoặc HLMDPH_178BD9035EDD_2
- file mode import failure: verify `fileFormat` matches your project settings
