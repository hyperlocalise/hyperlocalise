## Cấu hình bắt buộc

```jsonc
{
  "storage": {
    "adapter": "lilt",
    "config": {
      "projectID": "123",
      "apiTokenEnv": "LILT_API_TOKEN"
    }
  }
}
```

## Giải quyết token

1. biến môi trường có tên là`storage.config.apiTokenEnv`
2. fallback: `LILT_API_TOKEN`

`apiToken` inline in config is not supported. Use environment variables only.

## Các trường cấu hình tùy chọn

- `targetLanguages`
- `timeoutSeconds` (defaults to `30`)
- `pollIntervalMs` (defaults to `1000`)
- `maxPolls` (defaults to `60`)

## Hành vi kéo

- bắt đầu một công việc xuất khẩu cho mỗi định dạng
- trạng thái xuất phiếu khảo sát cho đến khi công việc hoàn thành
- tải xuống các tệp JSON (một tệp hoặc ZIP)
- tạo các bản ghi trở về dạng chuẩn và bỏ qua các hàng khóa/ngôn ngữ/giá trị trống

## Các vấn đề phổ biến

- Thiếu token: export`LILT_API_TOKEN` trong cùng một phiên shell
- Thiếu ID dự án: set `storage.config.projectID`
- timeout or slow export: increase `timeoutSeconds`, `pollIntervalMs`, or `maxPolls`
