## Cấu hình bắt buộc

```jsonc
{
  "storage": {
    "adapter": "lilt",
    "config": {
      "projectID": "123",
      "apiTokenEnv": "LILT_API_TOKEN"
    }
  }
}
```

## Phân giải token

1. biến môi trường có tên là `storage.config.apiTokenEnv`
2. fallback: `LILT_API_TOKEN`

`apiToken` nội tuyến trong cấu hình không được hỗ trợ. Chỉ sử dụng biến môi trường.

## Các trường cấu hình tùy chọn

- `targetLanguages`
- `timeoutSeconds` (mặc định là `30`)
- `pollIntervalMs` (mặc định là `1000`)
- `maxPolls` (mặc định là `60`)

## Hành vi kéo

- bắt đầu một công việc xuất cho từng ngôn ngữ
- kiểm tra trạng thái xuất khảo sát cho đến khi công việc hoàn tất
- tải xuống các artifact JSON (một tệp hoặc ZIP)
- chuẩn hóa các mục nhập và bỏ qua các hàng khóa/miền/ngôn ngữ/giá trị trống

## Các vấn đề thường gặp

- thiếu token: export `LILT_API_TOKEN` trong cùng phiên shell
- thiếu id dự án: đặt `storage.config.projectID`
- hết thời gian chờ hoặc xuất chậm: tăng `timeoutSeconds`, `pollIntervalMs`, hoặc `maxPolls`
