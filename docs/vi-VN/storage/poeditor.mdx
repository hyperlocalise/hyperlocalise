## Cấu hình bắt buộc

```jsonc
{
  "storage": {
    "adapter": "poeditor",
    "config": {
      "projectID": "123456",
      "apiTokenEnv": "POEDITOR_API_TOKEN"
    }
  }
}
```

## Giải quyết token

1. biến môi trường có tên là`storage.config.apiTokenEnv`
2. fallback: `POEDITOR_API_TOKEN`

## Các trường cấu hình tùy chọn

- `sourceLanguage`
- `targetLanguages`
- `timeoutSeconds`

## Các vấn đề thường gặp

- Token không hợp lệ: Kiểm tra xem biến môi trường có được xuất trong shell hiện tại hay không.
- ID dự án không đúng: Vui lòng xác minh ID dự án số từ POEditor.
- Thiếu thông tin ngôn ngữ: Đảm bảo `targetLanguages` phù hợp với các mục tiêu cấu hình
