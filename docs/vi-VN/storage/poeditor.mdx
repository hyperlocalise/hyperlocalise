## Cấu hình bắt buộc

```jsonc
{
  "storage": {
    "adapter": "poeditor",
    "config": {
      "projectID": "123456",
      "apiTokenEnv": "POEDITOR_API_TOKEN"
    }
  }
}
```

## Phân giải mã thông báo

1. biến môi trường được đặt tên bởi `storage.config.apiTokenEnv`
2. dự phòng: `POEDITOR_API_TOKEN`

## Các trường cấu hình tùy chọn

- `sourceLanguage`
- `targetLanguages`
- `timeoutSeconds`

## Các vấn đề thường gặp

- token không hợp lệ: hãy xác minh rằng biến môi trường đã được export trong shell hiện tại
- Sai ID dự án: xác minh ID dự án dạng số từ POEditor
- thiếu ngôn ngữ: đảm bảo `targetLanguages` phù hợp với các mục tiêu cấu hình
