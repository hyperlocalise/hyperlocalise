## Cấu hình bắt buộc

```jsonc
{
  "storage": {
    "adapter": "poeditor",
    "config": {
      "projectID": "123456",
      "apiTokenEnv": "POEDITOR_API_TOKEN"
    }
  }
}
```

## Phân giải token

1. Đặt biến môi trường `POEDITOR_API_TOKEN` (khuyến nghị)
2. Hoặc dùng một biến môi trường tùy chỉnh bằng cách đặt `apiTokenEnv` trong cấu hình

Nếu `apiTokenEnv` bị bỏ qua, nó sẽ mặc định là `POEDITOR_API_TOKEN`.

## Các trường cấu hình tùy chọn

- `sourceLanguage`
- `targetLanguages`
- `timeoutSeconds`

## Các vấn đề thường gặp

- token không hợp lệ: xác minh biến môi trường đã được export trong shell hiện tại
- ID dự án không đúng: hãy xác minh ID dự án dạng số từ POEditor
- thiếu locale: đảm bảo `targetLanguages` khớp với các mục tiêu cấu hình
