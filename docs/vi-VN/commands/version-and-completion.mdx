## Phiên bản

```bash
hyperlocalise version
```

Định dạng đầu ra mong đợi:

```text
hyperlocalise: <version>
```

## Hoàn tất

Sử dụng bản dựng sẵn-trong lệnh hoàn tất Cobra:

```bash
hyperlocalise completion --help
```

Sau đó tạo và cài đặt các script hoàn tất lệnh cho shell của bạn.
