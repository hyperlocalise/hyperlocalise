## Phiên bản

```bash
hyperlocalise version
```

Định dạng đầu ra dự kiến:

```text
hyperlocalise: <version>
```

## Cập nhật

Cập nhật lên bản phát hành mới nhất đã được công bố:

```bash
hyperlocalise update
```

Ghim một thẻ phát hành cụ thể:

```bash
hyperlocalise update v1.2.3
```

## Hoàn thành

Sử dụng lệnh hoàn thành tích hợp sẵn của Cobra:

```bash
hyperlocalise completion --help
```

Sau đó, tạo và cài đặt các script hoàn thành lệnh cho shell của bạn.
