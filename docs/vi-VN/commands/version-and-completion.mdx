## Phiên bản

```bash
hyperlocalise version
```

Định dạng đầu ra dự kiến:

```text
hyperlocalise: <version>
```

## Cập nhật

Cập nhật lên phiên bản mới nhất đã phát hành:

```bash
hyperlocalise update
```

Gắn một thẻ phiên bản cụ thể:

```bash
hyperlocalise update v1.2.3
```

## Hoàn thành

Sử dụng lệnh hoàn thành tích hợp của Cobra:

```bash
hyperlocalise completion --help
```

Sau đó, tạo và cài đặt các script hoàn thành cho shell của bạn.
