## Hình dạng giao diện dòng lệnh

```bash
hyperlocalise [command]
```

Các lệnh chính:

- `init`
- `run`
- `eval`
- `status`
- `sync pull`
- `sync push`
- `version`
- `update`
- `completion`

## Điểm khởi đầu theo nhiệm vụ

- First setup: [init](/commands/init)
- Tạo bản dự thảo cục bộ: [chạy](/commands/run)
- Thực hiện thí nghiệm và kiểm soát chất lượng: [eval](/commands/eval)
- Báo cáo tiến độ: [trạng thái](/commands/status)
- Lấy các bản dịch đã được chọn lọc: [đồng bộ lấy HLMDPH_BE53411E4A2A_0]
- Push các thay đổi cục bộ: [đồng bộ hóa đẩy](/commands/sync-push)
