## Hình dạng CLI

```bash
hyperlocalise [command]
```

Lệnh chính:

- `init`
- `run`
- `eval`
- `status`
- `sync pull`
- `sync push`
- `version`
- `completion`

## Điểm bắt đầu theo tác vụ

- Thiết lập lần đầu: [khởi tạo](/commands/init)
- Tạo bản nháp cục bộ: [chạy](/commands/run)
- Thử nghiệm chất lượng và cổng kiểm soát: [đánh giá](/commands/eval)
- Báo cáo tiến độ: [trạng thái](/commands/status)
- Kéo các bản dịch đã được chọn lọc: [đồng bộ hóa kéo về](/commands/sync-pull)
- Đẩy các thay đổi cục bộ: [đồng bộ đẩy](/commands/sync-push)
