## Hình dạng CLI

```bash
hyperlocalise [command]
```

Các lệnh chính:

- `init`
- `crowdin`
- `run`
- `check`
- `eval`
- `status`
- `sync pull`
- `sync push`
- `version`
- `update`
- `completion`

## Điểm bắt đầu theo tác vụ

- Thiết lập lần đầu: [init](/commands/init)
- Quy trình làm việc của tệp Crowdin: [crowdin](/commands/crowdin)
- Tạo bản nháp cục bộ: [run](/commands/run)
- Kiểm tra tính toàn vẹn bản dịch: [check](/commands/check)
- Thử nghiệm chất lượng và kiểm soát: [eval](/commands/eval)
- Đang báo cáo tiến độ: [status](/commands/status)
- Kéo các bản dịch đã được tuyển chọn: [sync pull](/commands/sync-pull)
- Đẩy các thay đổi cục bộ: [đồng bộ đẩy](/commands/sync-push)
