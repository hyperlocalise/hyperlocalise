title: "Khởi tạo"
description: "Tạo hoặc ghi đè tệp mẫu i18n.jsonc trong dự án của bạn."
---

## Sử dụng

```bash
hyperlocalise init [--force]
```

## Nó làm gì

- viết `i18n.jsonc` từ mẫu mặc định
- fails if file already exists (unless `--force`)

## Cờ

- `--force`: overwrite an existing `i18n.jsonc`

## Luồng hoạt động điển hình

```bash
hyperlocalise init
hyperlocalise run --dry-run
```
