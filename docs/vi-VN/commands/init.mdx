title: "khởi tạo"
description: "Tạo hoặc ghi đè mẫu i18n.jsonc trong dự án của bạn."
---

## Cách sử dụng

```bash
hyperlocalise init [--force]
```

## Nó làm gì

- writes `i18n.jsonc` từ bản dựng-trong mẫu
- thất bại nếu tệp đã tồn tại (trừ khi `--force`)

## Gắn cờ

- `--force`: ghi đè lên một mục hiện có `i18n.jsonc`

## Luồng điển hình

```bash
hyperlocalise init
hyperlocalise run --dry-run
```
