## Kiểm tra chất lượng

```bash
make fmt
make lint
make test
make precommit
```

## Luồng phát hành

```bash
git tag v0.1.0
git push origin v0.1.0
```

Các tệp tự động hóa phát hành:

- `.github/workflows/release.yml`
- `.goreleaser.yml`

## Các mục tiêu tạo phẩm đã xuất bản

- macOS: `amd64`, `arm64`
- Linux: `amd64`, `arm64`
- các gói: `.deb`, `.rpm`
