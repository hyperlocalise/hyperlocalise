title: "Bảo trì bản phát hành"
description: "Quy trình quản lý để kiểm tra chất lượng và xuất bản phiên bản bằng CLI."
---

## Kiểm tra chất lượng

```bash
make fmt
make lint
make test
make precommit
```

## Quy trình phát hành

```bash
git tag v0.1.0
git push origin v0.1.0
```

Các tệp tự động hóa phát hành:

- `.github/workflows/release.yml`
- `.goreleaser.yml`

## Đích đến của artifact đã xuất bản

- macOS: `amd64`, `arm64`
- Linux: `amd64`, `arm64`
- packages:`.deb`, `.rpm`
