## Kiểm tra chất lượng

```bash
make fmt
make lint
make test
make precommit
```

## Luồng phát hành

```bash
git tag v0.1.0
git push origin v0.1.0
```

Tệp tự động hóa phát hành:

- `.github/workflows/release.yml`
- `.goreleaser.yml`

## Các mục tiêu tạo tác đã xuất bản

- macOS: `amd64`, `arm64`
- Linux: `amd64`, `arm64`
- gói: `.deb`, `.rpm`
