## Điều kiện tiên quyết

- Go 1.26+
- `git`
- `make` có sẵn trong shell của bạn

## Sao chép kho lưu trữ

```bash
git clone https://github.com/quiet-circles/hyperlocalise.git
cd hyperlocalise
```

## Công cụ và mô-đun Bootstrap

```bash
make bootstrap
```

Thao tác này tải xuống các mô-đun Go và cài đặt bộ công cụ lint đã được ghim phiên bản.

## Vòng lặp phát triển hằng ngày

```bash
make fmt
make lint
make test
make check-build
```

## Chạy CLI cục bộ

```bash
make run
# or
go run -ldflags "-X main.version=dev" main.go
```

## Cài đặt tệp nhị phân cục bộ

```bash
make install
hyperlocalise version
```

## Vị trí mã khóa

- `cmd/`: Lệnh Cobra, cờ và hành vi đầu ra CLI
- `internal/config/`: `i18n.jsonc` phân tích cú pháp và xác thực
- `internal/i18n/runsvc/`: `run` Luồng lập kế hoạch và thực thi
- `internal/i18n/syncsvc/`: điều phối đồng bộ hóa và logic xung đột
- `internal/i18n/storage/`: hợp đồng adapter và tích hợp nhà cung cấp

## Trước khi mở một PR

```bash
make precommit
```

Lệnh này chạy định dạng, linting, kiểm thử và kiểm tra build theo cùng một quy trình mà các cộng tác viên sử dụng.
