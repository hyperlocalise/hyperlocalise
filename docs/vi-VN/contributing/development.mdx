## Các điều kiện tiên quyết

- Go 1.26+
- `git`
- `make` available in your shell

## Sao chép kho lưu trữ

```bash
git clone https://github.com/hyperlocalise/hyperlocalise.git
cd hyperlocalise
```

## Công cụ và mô-đun Bootstrap

```bash
make bootstrap
```

Chức năng này tải xuống các mô-đun Go và cài đặt công cụ kiểm lint đã được chỉ định.

## Vòng lặp phát triển hàng ngày

```bash
make fmt
make lint
make test
make check-build
```

## Chạy CLI cục bộ

```bash
make run
# or
go run -ldflags "-X main.version=dev" ./apps/cli
```

## Cài đặt bản nhị phân cục bộ

```bash
make install
hyperlocalise version
```

## Vị trí các mã khóa

- `apps/cli/cmd/`: Cobra commands, flags, and CLI output behavior
- `pkg/i18nconfig/`: `i18n.jsonc` parsing and validation
- `apps/cli/internal/i18n/runsvc/`: `run` planning and execution flow
- `apps/cli/internal/i18n/syncsvc/`: sync orchestration and conflict logic
- `internal/i18n/storage/`: hợp đồng và tích hợp nhà cung cấp

## Trước khi mở một PR

```bash
make precommit
```

Điều này thực hiện định dạng, kiểm tra, chạy thử và kiểm tra xây dựng theo quy trình được sử dụng bởi các nhà đóng góp.
