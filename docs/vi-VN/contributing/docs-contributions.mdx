## Sửa đổi trên GitHub

1. mở trang
2. nhấp vào chỉnh sửa,
3. gửi một yêu cầu rút gọn.

## Quy trình tài liệu nội bộ

1. Sao chép kho lưu trữ,
2. install Mintlify CLI (`npm i -g mint`),
3. chạy HLMDPH_E2F2CF3B2B5E_0 trong HLMDPH_A5BD43A1D97E_1,
4. validate links with `mint broken-links`,
5. mở yêu cầu hợp nhất.

## Tiêu chuẩn viết

- Sử dụng thì hiện tại và ngôi thứ hai.
- giữ các ví dụ có thể thực thi
- Bắt đầu mỗi trang bằng mục tiêu và các điều kiện tiên quyết.
- đảm bảo các tùy chọn lệnh phù hợp với đầu ra trợ giúp CLI hiện tại.
