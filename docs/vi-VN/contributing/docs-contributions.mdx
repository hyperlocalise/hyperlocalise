## Chỉnh sửa trên GitHub

1. mở trang,
2. nhấp vào Chỉnh sửa,
3. gửi một pull request.

## Quy trình tài liệu cục bộ

1. sao chép kho lưu trữ,
2. cài đặt Mintlify CLI (`npm i -g mint`),
3. chạy `mint dev` trong `docs/`,
4. xác thực liên kết với `mint broken-links`,
5. mở pull request.

## Tiêu chuẩn viết

- Hãy dùng thể chủ động và ngôi thứ hai,
- giữ cho các ví dụ có thể chạy được,
- Mở đầu mỗi trang bằng mục tiêu và các điều kiện tiên quyết,
- giữ các cờ lệnh khớp với đầu ra trợ giúp CLI hiện tại.
