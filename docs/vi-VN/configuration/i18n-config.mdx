## Tổng quan về tệp

`i18n.jsonc` controls locales, file mapping, model profiles, and rule selection.

## Các khu vực

- `locales.source`: nguồn vị trí.
- `locales.targets`: list of target locales.
- optional `locales.fallbacks`: fallback order per locale.

## Thau

`buckets` map source files to target output templates.

Mỗi bản ánh xạ sử dụng:

- `from`: source path template
- `to`: mẫu đường dẫn đích

Use `{{source}}`, `{{target}}`, and `{{localeDir}}` in templates. `{{localeDir}}` resolves to an empty segment when target equals source, and to the target locale otherwise.


### Mẫu định dạng tệp CSV

Sử dụng bản đồ tệp bucket để mô hình hóa quy trình CSV.

Các tệp theo ngữ cảnh:

```json
{
  "from": "translations/en.csv",
  "to": "translations/[locale].csv"
}
```

Tệp đa ngôn ngữ được chia sẻ:

```json
{
  "from": "translations/catalog.csv",
  "to": "translations/catalog.csv"
}
```

For shared multi-locale CSV files, keep one key column and one target locale column that `run` can update consistently.

## Nhóm

`groups.<name>` defines what to process together.

- `targets`: subset of `locales.targets`
- `buckets`: list of bucket names

## Các hồ sơ LLM

`llm.profiles.<name>` trường:

- `provider`: `openai`, `azure_openai`, `anthropic`, `gemini`, `bedrock`, `lmstudio`, `groq`, or `ollama`
- `model`: mã mô hình nhà cung cấp
- `system_prompt` (tùy chọn): mẫu tin nhắn hệ thống
- `user_prompt` (optional): explicit user-message template
- `prompt` (deprecated, optional): legacy prompt template used as system fallback

Các biến đầu vào:

- `{{source}}`
- `{{target}}`
- `{{input}}`

## Quy tắc

`llm.rules` choose profile per group.

- `priority`: người thắng
- `group`: tên nhóm
- `profile`: profile name

## Bộ nhớ đệm

`cache` configures local L1/L2 behavior.

- `enabled` (tùy chọn): bật dịch vụ bộ nhớ đệm.
- `db_path` (tùy chọn): đường dẫn sqlite cho cơ sở dữ liệu cache.
- `sqlite` (tùy chọn): điều chỉnh kết nối.
- `l1.max_items` (tùy chọn): Giới hạn loại bỏ bộ nhớ cấp 1.
- `l2.enabled` (optional): enable translation memory lookup/upsert.
- `l2.auto_accept_threshold` (optional): fuzzy auto-accept threshold in `[0,1]` (default `0.90`).

Danh sách giá trị enum cho TM metadata:

- nguồn gốc: `curated` | `draft` | `llm` | `tms` | `unknown`
- source: `run` | `sync_pull` | `sync_push` | `manual` | `import` | `legacy` | `unknown`

## Cấu hình ban đầu

```bash
hyperlocalise init
```

Then edit the generated `i18n.jsonc` for your project.

## Mẹo kiểm tra

- Keep group targets inside `locales.targets`.
- Keep group buckets aligned with `buckets` keys.
- Giữ nguyên tên hồ sơ giữa `profiles` và `rules`.
