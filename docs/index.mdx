`hyperlocalise` is a Go CLI for localization workflows that combine local files, LLM draft generation, and remote TMS curation.

## Features

- Fast CLI with Go + Cobra command structure.
- Local translation generation with lock-file based task skipping.
- Sync workflows for POEditor, Lokalise, and Crowdin adapters.
- Sidecar metadata model for translation provenance (`draft` vs `curated`).
- Release pipeline with multi-arch binaries and Linux packages.

## Core commands

```bash
hyperlocalise init
hyperlocalise run
hyperlocalise status
hyperlocalise sync pull
hyperlocalise sync push
```
